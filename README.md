# salesforce-contact-management-flow

## Descrição

Screen Flow desenvolvido no Salesforce para gerenciamento inteligente de contatos.

O fluxo verifica se já existe um contato com o e-mail informado. Caso exista, permite atualizar os dados. Caso contrário, cria um novo contato.

## Funcionalidades

* Busca de contatos por e-mail
* Prevenção de registros duplicados
* Criação de novos contatos
* Atualização de contatos existentes
* Interface com Screen Flow
